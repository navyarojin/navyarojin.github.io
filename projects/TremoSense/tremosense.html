TremoSense

Wearable system to measure tremor frequency and amplitude for Parkinson’s medication efficacy analysis

← Back to Projects

Overview

TremoSense is a custom embedded system designed to capture and analyze tremor characteristics such as frequency and amplitude from the wrist. The goal of this device is to assist in evaluating medication effectiveness in Parkinson’s disease patients.

The system integrates motion sensing, signal processing, and data acquisition on a compact PCB suitable for wearable applications.

Key Features

Motion Sensing

High-resolution IMU for detecting fine hand tremors.

Tremor Analysis

Captures tremor frequency and amplitude from wrist motion.

Wearable Design

Compact PCB suitable for continuous patient monitoring.

Microcontroller Based

Low-power MCU for real-time data acquisition and processing.

Data Logging

Supports external storage / serial data extraction.

Healthcare Focus

Designed specifically for Parkinson’s tremor assessment.
