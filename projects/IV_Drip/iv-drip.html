IV Drip Monitoring System

Non-invasive IV drip monitoring system using optical sensing, flow detection, and alerting — built using Arduino Nano with a custom 3D-printed enclosure.

Overview

The system uses an optical IR sensing method to track drop rate in IV drip chambers. It warns clinicians when flow rate deviates or the IV bag is nearly empty. The design focuses on reliability, low cost, and clinical usability.

Key Features

  • Non-invasive optical sensing of fluid drop rate
  • Arduino Nano used for real-time processing
  • Visual + audio alerts for low-flow or empty bag
  • 3D-printed enclosure optimized for hospital use
  • Low-cost and scalable design for high-volume deployment
