Overview

This project is a fully custom ESP32-S3 development board designed for embedded prototyping, healthcare electronics, and IoT applications. The board integrates power management, sensing, real-time clock functionality, and expandable storage on a compact PCB.

Key Features

ESP32-S3

ESP32-S3-MINI module for Wi-Fi enabled embedded applications.

Power Management

USB power + Li-ion battery charging using LTC4054 with LDO-regulated 3.3 V output.

USB-C Interface

USB Type-C with ESD protection for power and data.

Motion & Time

MPU6050 IMU and DS3231 high-accuracy RTC.

Storage

SPI-based micro-SD card interface for data logging.

Debug & IO

UART, I2C, SPI and GPIO headers for expansion.
